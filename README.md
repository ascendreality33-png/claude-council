# Claude Council

Claude Council is a multi-agent framework built on top of Claude, Anthropic's AI model. It enables multiple Claude agents to collaborate, delegate tasks, and reason collectively to solve complex problems that benefit from parallel or specialized thinking.

## Overview

- Orchestrate multiple Claude agents working in parallel or in sequence
- Assign specialized roles to individual agents (e.g., researcher, critic, planner, executor)
- Aggregate and synthesize outputs across agents for higher-quality results
- Flexible architecture for building custom multi-agent pipelines
